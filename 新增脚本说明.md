## 🔧 使用方法

所有脚本都可以独立运行：

```bash
cd ~/Desktop/VSCodePyScripts/NEWS

# 测试单个脚本
python3 fetch_baidu.py
python3 fetch_zhihu.py
python3 fetch_jin10.py

# 测试所有新脚本
for script in fetch_baidu.py fetch_zhihu.py fetch_jin10.py fetch_36kr.py fetch_wallstreetcn.py fetch_bilibili.py fetch_toutiao.py fetch_thepaper.py fetch_ithome.py fetch_juejin.py fetch_github.py fetch_v2ex.py fetch_weibo.py; do
    echo "Testing $script..."
    python3 $script
    echo ""
done
```

## ⚠️  注意事项

1. **部分API需要Cookie或认证**
   - 微博需要有效的Cookie
   - 部分网站可能有反爬虫机制

2. **API稳定性**
   - 某些API可能随时变化
   - 建议定期检查脚本是否正常工作

3. **请求频率**
   - 建议控制请求频率，避免被封IP
   - 可以考虑添加随机延迟

4. **未完全测试的脚本**
   - 标记为 ⚠️  的脚本建议先测试再使用
   - 可能需要根据实际情况调整
